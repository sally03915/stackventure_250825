
- ③ /27일 경우 (30개 사용 가능)  
- ④ /27보다 더 큰 서브넷이어야 가능함  
🧸 **쉬운 해설:**  
주소는 집 주소 같은 거야. 첫 번째 방은 '건물 이름', 마지막 방은 '모두에게 알림'이라서 못 써.
